利用規約

制定日：2026年4月30日　最終更新日：2026年4月30日

本利用規約（以下「本規約」）は、AOリサーチ（以下「本サービス」）の利用条件を定めるものです。ユーザーは本規約に同意のうえ、本サービスをご利用ください。

第1条（適用）

本規約は、ユーザーと運営者（新里琉）との間の本サービスに関わる一切の関係に適用されます。

第2条（利用登録）

登録希望者が所定の方法で登録申請を行い、運営者がこれを承認することによって利用登録が完了します。運営者は以下の場合に登録申請を承認しないことがあります。

  • 虚偽の情報を申告した場合
  • 本規約に違反したことがある者からの申請である場合
  • その他、運営者が不適当と判断した場合

第3条（禁止事項）

ユーザーは以下の行為をしてはなりません。

  • 法令または公序良俗に違反する行為
  • 本サービスを通じて得た情報の無断転載・商業利用
  • 本サービスの運営を妨害する行為
  • 他のユーザーまたは第三者への不正アクセス
  • アカウントの第三者への譲渡・貸与
  • リバースエンジニアリング、スクレイピングその他本サービスの技術的解析
  • その他、運営者が不適当と判断した行為

第4条（料金・支払い）

有料プランの利用料金は、各プランページに表示された金額とします。支払いは Stripe を通じたクレジットカード払いとし、月次で自動更新されます。ユーザーは解約しない限り翌月分の料金が請求されます。

第5条（キャンセル・返金）

サブスクリプションはいつでもキャンセルできます。キャンセル後は当該請求期間の終了まで有料機能を利用できます。既にお支払いいただいた料金の返金は原則として行いません。ただし、システム障害等により本サービスを全く利用できなかった期間については個別に対応します。

第6条（本サービスの提供の停止等）

運営者は以下の事由によりサービス提供を停止・中断できます。停止・中断によりユーザーに生じた損害について、運営者は責任を負いません。

  • サーバーの保守・点検
  • 天災・停電等の不可抗力
  • 外部 API の障害や仕様変更
  • その他、運営者が停止・中断を必要と判断した場合

第7条（免責事項）

本サービスが提供する情報は、公開情報を AI で収集・整理したものです。情報の正確性・完全性・最新性を保証するものではありません。本サービスの利用によって生じた損害（入試結果を含む）について、運営者は一切の責任を負いません。入試に関する最終的な判断は必ず大学公式情報でご確認ください。

第8条（知的財産権）

本サービスのシステム・デザイン・コンテンツに関する知的財産権は運営者に帰属します。ユーザーが入力した情報の権利はユーザーに帰属しますが、サービス改善目的での匿名利用を許諾するものとします。

第9条（プライバシー）

個人情報の取り扱いについてはプライバシーポリシーをご覧ください。

第10条（規約の変更）

運営者は必要に応じて本規約を変更できます。変更後の規約はサービス上に掲載された時点で効力を生じます。変更後も本サービスを利用し続けた場合、変更後の規約に同意したものとみなします。

第11条（準拠法・管轄）

本規約は日本法に準拠します。本規約に関する紛争は、運営者の住所地を管轄する裁判所を第一審の専属合意管轄裁判所とします。
